# BI Platform - Платформа анализа успеваемости студентов

Современная платформа для визуализации и анализа успеваемости студентов на кафедре с применением BI-инструментов. Разработана с использованием React, TypeScript и Recharts для создания интерактивных дашбордов и графиков.

## 🚀 Возможности

### Основные функции

- **Интерактивные дашборды** - Создание персонализированных дашбордов с различными виджетами
- **Множественные типы графиков**:
  - Линейные графики (динамика среднего балла)
  - Столбчатые диаграммы (успеваемость по группам, распределение по оценкам)
  - Круговые диаграммы (успеваемость по предметам)
  - Областные графики (тренд успеваемости)
- **Метрики и KPI** - Отображение ключевых показателей успеваемости:
  - Средний балл по кафедре
  - Общее количество студентов
  - Процент успеваемости
  - Количество отличников
- **Таблицы данных** - Интерактивные таблицы с информацией о студентах, их оценках и средних баллах
- **Фильтры и анализ**:
  - Фильтрация по периодам (7, 30, 90 дней, семестр)
  - Фильтрация по предметам (Математика, Физика, Информатика, Химия, Английский)
  - Фильтрация по группам (ИТ-21-01, ИТ-21-02, ИТ-22-01, ИТ-22-02, ИТ-23-01)
- **Drill-down функциональность** - Детализация данных при клике на элементы графиков
- **Экспорт данных** - Экспорт таблиц с данными о студентах в CSV формат
- **Адаптивный дизайн** - Полностью адаптивный интерфейс для всех устройств

### Технические особенности

- ⚡️ **Vite** - Быстрая сборка и разработка
- ⚛️ **React 18** - Современный UI фреймворк
- 📘 **TypeScript** - Типобезопасность
- 📊 **Recharts** - Мощная библиотека для визуализации данных
- 🎨 **Tailwind CSS** - Современная стилизация
- 🎯 **Lucide React** - Иконки

## 📦 Установка

1. **Клонируйте репозиторий** (или используйте текущую директорию)

2. **Установите зависимости**:
```bash
npm install
```

## 🛠️ Запуск проекта

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для production

```bash
npm run build
```

Собранные файлы будут находиться в директории `dist/`

### Превью production сборки

```bash
npm run preview
```

## 🎨 Компоненты

### Dashboard
Главный компонент дашборда, объединяющий все виджеты и компоненты визуализации.

### Charts

- **MetricCard** - Карточка с метрикой и трендом
- **LineChart** - Линейный график для временных рядов
- **BarChart** - Столбчатая диаграмма для сравнения категорий
- **PieChart** - Круговая диаграмма с поддержкой drill-down
- **AreaChart** - Областной график для визуализации трендов
- **DataTable** - Интерактивная таблица с сортировкой и экспортом

### Filters
Компонент фильтрации данных по периоду, категории и региону.

## 🔧 Настройка

### Цветовая схема

Цвета настраиваются в `tailwind.config.js`:

```js
colors: {
  primary: {
    // Цветовая палитра
  }
}
```

### Данные

Мок-данные находятся в `src/data/mockData.ts`. Для подключения реального API замените функцию `generatePerformanceData` на вызов API. Данные включают:
- Динамику успеваемости студентов по дням
- Данные по предметам (средний балл)
- Данные по группам (средний балл)
- Распределение по оценкам
- Детальную информацию о студентах (ФИО, группа, предмет, оценка, средний балл, семестр)

## 📝 Расширение функциональности

### Добавление нового типа графика

1. Создайте компонент в `src/components/charts/`
2. Используйте Recharts для визуализации
3. Добавьте компонент в `Dashboard.tsx`

### Подключение реального API

1. Создайте сервис для работы с API в `src/services/`
2. Замените мок-данные на реальные запросы
3. Добавьте обработку ошибок и загрузки

### Добавление новых фильтров

Расширьте компонент `Filters.tsx` новыми опциями фильтрации.
