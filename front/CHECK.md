# Проверка работоспособности

Если сайт ничего не показывает, выполните следующие шаги:

## 1. Убедитесь, что зависимости установлены

```bash
npm install
```

## 2. Очистите кэш и переустановите зависимости (если нужно)

```bash
rm -rf node_modules
rm -rf dist
npm install
```

## 3. Запустите dev-сервер

```bash
npm run dev
```

## 4. Проверьте консоль браузера

Откройте DevTools (F12) и проверьте вкладку Console на наличие ошибок.

## 5. Проверьте вкладку Network

Убедитесь, что все файлы загружаются успешно (статус 200).

## Возможные проблемы:

- **Белый экран** - проверьте консоль браузера на ошибки
- **404 ошибки** - убедитесь, что dev-сервер запущен
- **Порт занят** - измените порт в vite.config.ts (сейчас 3000)

## Если ничего не помогает:

Попробуйте пересобрать проект:

```bash
npm run build
npm run preview
```
